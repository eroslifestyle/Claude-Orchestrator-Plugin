# Git Clone Installation Instructions

This document provides instructions for installing the Claude Orchestrator Plugin via direct git clone.

---

## Option 1: Clone Directly into ~/.claude/

### Linux / macOS

```bash
# Backup existing .claude directory if it exists
if [ -d ~/.claude ]; then
  mv ~/.claude ~/.claude.backup.$(date +%Y%m%d%H%M%S)
fi

# Clone directly into ~/.claude/
cd ~ && git clone https://github.com/eroslifestyle/Claude-Orchestrator-Plugin.git .claude
```

### Windows (PowerShell)

```powershell
# Backup existing .claude directory if it exists
if (Test-Path "$env:USERPROFILE\.claude") {
    $timestamp = Get-Date -Format "yyyyMMddHHmmss"
    Rename-Item "$env:USERPROFILE\.claude" "$env:USERPROFILE\.claude.backup.$timestamp"
}

# Clone directly into ~/.claude/
Set-Location $env:USERPROFILE
git clone https://github.com/eroslifestyle/Claude-Orchestrator-Plugin.git .claude
```

## Option 2: Clone to Temp and Copy

Use this method if you have existing Claude configuration you want to preserve or merge manually.

### Linux / macOS

```bash
# Clone to a temporary directory
TEMP_DIR=$(mktemp -d)
git clone https://github.com/eroslifestyle/Claude-Orchestrator-Plugin.git "$TEMP_DIR/orchestrator"

# Create .claude directory if it doesn't exist
mkdir -p ~/.claude

# Copy orchestrator skill
cp -r "$TEMP_DIR/orchestrator" ~/.claude/skills/orchestrator

# Copy agents definitions (if you want them)
cp -r "$TEMP_DIR/orchestrator/agents" ~/.claude/agents

# Cleanup
rm -rf "$TEMP_DIR"

echo "Installation complete!"
```

### Windows (PowerShell)

```powershell
# Clone to a temporary directory
$tempDir = [System.IO.Path]::GetTempPath()
$tempPath = Join-Path $tempDir "orchestrator-install"
New-Item -ItemType Directory -Force -Path $tempPath | Out-Null

git clone https://github.com/eroslifestyle/Claude-Orchestrator-Plugin.git "$tempPath\orchestrator"

# Create .claude directory if it doesn't exist
$claudeDir = "$env:USERPROFILE\.claude"
if (-not (Test-Path $claudeDir)) {
    New-Item -ItemType Directory -Force -Path $claudeDir | Out-Null
}

# Create skills directory if it doesn't exist
$skillsDir = "$claudeDir\skills"
if (-not (Test-Path $skillsDir)) {
    New-Item -ItemType Directory -Force -Path $skillsDir | Out-Null
}

# Copy orchestrator skill
Copy-Item -Recurse -Force "$tempPath\orchestrator" "$skillsDir\orchestrator"

# Copy agents definitions (optional)
if (-not (Test-Path "$claudeDir\agents")) {
    New-Item -ItemType Directory -Force -Path "$claudeDir\agents" | Out-Null
}
Copy-Item -Recurse -Force "$tempPath\orchestrator\agents\*" "$claudeDir\agents\"

# Cleanup
Remove-Item -Recurse -Force $tempPath

Write-Host "Installation complete!" -ForegroundColor Green
```

## Option 3: Clone and Selectively Merge

If you have existing Claude configuration and want to selectively merge:

### Step 1: Clone to a separate directory

```bash
# Linux/macOS
git clone https://github.com/eroslifestyle/Claude-Orchestrator-Plugin.git ~/orchestrator-temp

# Windows PowerShell
git clone https://github.com/eroslifestyle/Claude-Orchestrator-Plugin.git "$env:USERPROFILE\orchestrator-temp"
```

### Step 2: Review and copy what you need

```bash
# Example: Copy only the skill
cp -r ~/orchestrator-temp ~/.claude/skills/orchestrator

# Example: Copy only specific agents
cp ~/orchestrator-temp/agents/*.md ~/.claude/agents/
```

### Step 3: Cleanup

```bash
rm -rf ~/orchestrator-temp
```

## Post-Installation Steps

1. **Enable Agent Teams** (optional but recommended):
   ```bash
   # Add to ~/.claude/settings.json
   {
     "env": {
       "CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS": "1"
     }
   }
   ```

2. **Restart Claude Code** to pick up the new skill

3. **Verify installation**:
   ```
   /orchestrator
   ```

## Troubleshooting

### Error: "Destination path already exists"

The `.claude` directory already contains files. Use Option 2 or 3 instead.

### Error: "Permission denied"

Make sure you have write permissions to your home directory:
```bash
# Linux/macOS
chmod 755 ~

# Windows: Run PowerShell as Administrator
```

### Git not found

Install git first:
- **Windows**: `winget install Git.Git` or download from https://git-scm.com
- **macOS**: `xcode-select --install` or `brew install git`
- **Linux**: `sudo apt install git` or `sudo dnf install git`

---

## Uninstallation

To remove the orchestrator plugin:

```bash
# Remove only the skill
rm -rf ~/.claude/skills/orchestrator

# Remove everything (if you cloned directly)
rm -rf ~/.claude
# Then restore from backup if needed
```
